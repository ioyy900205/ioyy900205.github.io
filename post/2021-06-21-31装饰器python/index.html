一个装饰器的例子，很方便理解

@TOC

标题

先定义一个简单的函数foo

def foo():
    print 'call foo()'

现在我们想要计算次函数执行的时间，于是修改代码如下

import time

def foo():
    start = time.clock()
    print 'call foo()'
    end = time.clock()
    print 'using time:',end - start

然后有一个更快的方法

import time
 
def foo():
    print 'call foo()'
 
def cal_time(func):
    start = time.clock()
    func()
    end =time.clock()
    print 'using time:', end - start
 
cal_time(foo)

但是这样比较麻烦，如果foo换了，还得重新写其他的例如foo_1

import time
 
def foo():
    print 'call foo()'
 
# 定义一个计时器，传入一个函数，并返回另一个附加了计时功能的方法
def cal_time(func):
     
    # 定义一个内嵌的包装函数，给传入的函数加上计时功能的包装
    def wrapper():
        start = time.clock()
        func()
        end =time.clock()
        print 'using time:', end - start
     
    # 将包装后的函数返回
    return wrapper
 
foo = cal_time(foo)
foo()

再简化一点 其实 foo = cal_time(foo) 可以写成


@cal_time 
def foo()

于是

import time
 
def cal_time(func):
    def wrapper():
        start = time.clock()
        func()
        end =time.clock()
        print 'using time:', end - start
    return wrapper
 
@cal_time               #定义函数时加上这一行与另外写foo = cal_time(func):完全等价
def foo():
    print 'call foo()'
 
foo()
